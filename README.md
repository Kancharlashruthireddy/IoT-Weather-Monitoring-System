# IoT-Based Smart Weather Monitoring and Environmental Alert System

## Overview

This project monitors environmental parameters such as temperature, humidity, air quality, and rainfall. The system stores sensor data, generates alerts, and supports future cloud integration.

## Features

* Temperature Monitoring
* Humidity Monitoring
* Air Quality Monitoring
* Rainfall Detection
* Data Logging
* Alert Generation

## Hardware Components

* ESP32
* DHT11 Sensor
* MQ135 Air Quality Sensor
* Rain Sensor
* WiFi Module

## Software

* Python
* CSV Database
* GitHub

## Future Scope

* ThingSpeak Integration
* Mobile Dashboard
* Real-Time Notifications
