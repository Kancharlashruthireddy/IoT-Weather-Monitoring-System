# IoT Weather Monitoring System

## Introduction
This project monitors temperature, humidity, and air quality using IoT concepts.

## Objective
To develop a smart environmental monitoring system for real-time weather observation.

## Components Used
- ESP32
- DHT11 Sensor
- MQ135 Air Quality Sensor
- WiFi Module

## Working Principle
The sensors collect environmental data. The system processes the data and stores it for analysis and monitoring.

## Applications
- Smart Agriculture
- Weather Stations
- Environmental Monitoring
- Smart Cities

## Future Scope
- Cloud Integration
- Mobile Application
- Real-Time Alerts

## Conclusion
This project provides a reliable solution for monitoring environmental conditions using IoT technology.
